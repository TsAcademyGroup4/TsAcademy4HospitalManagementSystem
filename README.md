Hospital Management System Project

The Hospital Management System is a backend application developed as an academic project using Node.js and MongoDB. The system helps manage core hospital operations such as patient records, doctor information, appointment scheduling etc., through a centralized platform.

This project demonstrates backend development, RESTful APIs, and NoSQL database design while following software engineering principles.

Features:
Patient registration and management
Doctor and staff management
Appointment scheduling
Medical records management

Prerequisites:
Make sure you have the following installed:
Node.js (v18 or above)

How to Run the Project
1. Clone the Repository
git clone https://github.com/your-username/TsAcademy4HospitalManagementSystem.git
cd TsAcademy4HospitalManagementSystem

2. Install Dependencies
npm install

3. Configure Environment Variables
Create a .env file in the root directory and add:
PORT=5000
MONGO_URI=your_mongodb_connection_string

4. Start the Server
npm start

The server will start on:
http://localhost:5000
